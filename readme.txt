Enhanced dashboard widget with server information, AJAX refresh, and JSON export capabilities for support teams.

== Description ==

GC Support Info provides a comprehensive dashboard widget that displays crucial server and WordPress information for support teams and administrators. The plugin offers real-time data refresh capabilities and JSON export functionality for easy sharing with support staff.

**Key Features:**

* **Server Information**: Hostname, IP address, base URL, PHP version, and server load
* **Database Details**: Version, size, and table count
* **WordPress Information**: Version, active theme, parent theme detection, plugin count, and multisite status
* **AJAX Refresh**: Real-time data updates without page reload
* **JSON Export**: Export all data for easy sharing with support teams
* **GitHub Updates**: Automatic updates delivered via GitHub releases

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gc-support-info` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The support information widget will automatically appear on your dashboard.

That's it! No configuration needed - the widget will immediately start displaying your server information.

== Frequently Asked Questions ==

= Does this plugin require any configuration? =

No configuration is required. Simply activate the plugin and the support information widget will appear on your WordPress dashboard.

= Can I export the support information? =

Yes! Click the "Export JSON" button in the widget to download all support information as a JSON file that you can share with support teams.

= How often is the information updated? =

The information is updated in real-time when you click the "Refresh" button, or whenever you reload the dashboard page.

= Is this plugin compatible with multisite installations? =

Yes, the plugin works on both single-site and multisite WordPress installations and will indicate whether multisite is enabled.
